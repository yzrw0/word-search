# Word Search Game 🎮

A fun **Word Search Game** written in **C++**, playable in **Single Player** or **Multiplayer (vs Computer)** mode.  
The game generates a random 12x12 letter grid and challenges the player to find hidden words from different categories.

---

## 📝 Project Description

- Choose between **Single Player** or **Multiplayer (VS Computer)**.  
- Select a word category: **Programming**, **Fruits**, **Animals**, **Countries**.  
- Find 4 hidden words in a 12x12 random letter grid.  
- Track your score based on word length and difficulty.  
- Play against the computer for added challenge.

---

## 🎯 Features

- Random 12x12 grid generated every game.  
- Words placed **horizontally** or **vertically**.  
- Colored letters and scoreboard (Windows console colors).  
- Case-insensitive input.  
- Scoring system with Easy, Medium, Hard points.  
- Multiplayer mode with simple computer AI.

---

## 💻 Technologies Used

- **C++**  
- **Windows.h** library for colors  
- Random word placement using **rand()**
